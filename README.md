## 中文

本仓库是一个面向初中生的双语开源自学平台，利用 AI 技术（OpenAI, Gemini, Claude）生成高质量的理科（数学、物理、化学、生物）教学与自学内容，并以交互式网站的形式呈现。

### 项目初衷
- **地域针对性**：主要针对**北京**初中学生（以**人教版**教材大纲为核心）。
- **家庭自用**：最初为个人家庭辅导使用，现以 MIT 协议开源共享。
- **AI 赋能**：所有内容均由主流大模型生成，并经过人工基础校验。

### 技术栈
- **框架**：Next.js 14 (App Router) + TypeScript
- **样式**：Tailwind CSS（粉紫色调主题）
- **数学公式**：KaTeX（支持分数、次方、开方、几何符号等，与课本显示一致）
- **内容格式**：YAML（课程结构和学习内容）

### 项目结构
```
website/
├── src/data/
│   ├── curriculum.yaml                   # 全站课程菜单结构
│   └── math/grade7/
│       └── rational-numbers.yaml         # 有理数自学内容
├── src/components/                       # React 组件
├── src/lib/                              # 数据加载工具
└── src/app/                              # Next.js 页面
```

## English

A bilingual open-source self-study platform for middle school students, delivering AI-generated (OpenAI, Gemini, Claude) science curriculum materials through an interactive website.

### Mission
- **Target Audience**: Designed for students in **Beijing**, following the **PEP (人教版)** curriculum.
- **Personal Use**: Originally created for home tutoring, now shared under the MIT license.
- **AI-Powered**: Content is generated by state-of-the-art AI models and reviewed for accuracy.

### Tech Stack
- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS (pink & purple theme)
- **Math Rendering**: KaTeX (fractions, exponents, roots, geometry symbols)
- **Content Format**: YAML

### Quick Start
```bash
cd website
npm install
npm run dev
```
Visit http://localhost:3000
