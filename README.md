## 中文

本仓库是一个面向中小学生的双语开源自学平台，利用 AI 技术（OpenAI, Gemini, Claude）生成高质量的教学与自学内容，并以交互式网站的形式呈现。

### 项目初衷
- **家庭自用**：最初为个人家庭辅导使用，现以 MIT 协议开源共享。
- **AI 赋能**：所有内容均由主流大模型生成，并经过人工基础校验。
- **双语教学**：支持中文、英文、双语三种模式切换。

### 技术栈
- **框架**：Next.js 16 (App Router) + TypeScript，静态导出至 GitHub Pages
- **样式**：Tailwind CSS（柔和粉紫色调主题）
- **数学公式**：KaTeX（支持分数、次方、开方、几何符号等）
- **内容格式**：YAML（课程结构和学习内容）
- **语音**：OpenAI TTS API 预生成音频（词汇发音 + 听力对话）
- **进度追踪**：GitHub 私有仓库同步（Fine-grained PAT）或浏览器本地存储
- **评分系统**：每题对错记录、颜色分级徽章、分数历史

## English

A bilingual open-source self-study platform for primary and middle school students, delivering AI-generated (OpenAI, Gemini, Claude) curriculum materials through an interactive website.

### Mission
- **Personal Use**: Originally created for home tutoring, now shared under the MIT license.
- **AI-Powered**: Content is generated by state-of-the-art AI models and reviewed for accuracy.
- **Bilingual**: Supports Chinese, English, and dual-language display modes.

### Tech Stack
- **Framework**: Next.js 16 (App Router) + TypeScript, static export to GitHub Pages
- **Styling**: Tailwind CSS (soft pink & purple theme)
- **Math Rendering**: KaTeX (fractions, exponents, roots, geometry symbols)
- **Content Format**: YAML
- **Audio**: OpenAI TTS API for vocabulary pronunciation and listening dialogues
- **Progress Tracking**: GitHub private repo sync (Fine-grained PAT) or browser localStorage
- **Scoring**: Per-question right/wrong, color-coded badges, score history
